# Control de Inventario (Producción)

Aplicación web para control de stock en planta (diseño Tablet).

## Cómo iniciar
Esta aplicación usa Módulos ES6. Para abrirla, necesitas un servidor local simple (no funciona con solo doble clic en index.html por seguridad del navegador).

1. **Opción VS Code**: Click derecho en `index.html` -> "Open with Live Server".
2. **Opción Python**: Abre terminal aquí y ejecuta `python -m http.server`.
3. **Opción Node**: Abre terminal y ejecuta `npx serve`.

## Usuarios por defecto
- **Admin**: `admin` / `123`
- **Operario**: `operario` / `123`

## Personalización
Edita el archivo `js/config.js` para cambiar los tipos de prenda, colores, tallas o usuarios.
